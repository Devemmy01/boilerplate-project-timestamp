Timestamp Microservice

This API returns Unix and UTC timestamps for a given date string or Unix timestamp.

Usage

  • /api/ - Returns the current timestamp
  • /api/2015-12-25 - Returns the timestamp for a specific date
  • /api/1451001600000 - Returns the timestamp for a Unix time in milliseconds

Example Output

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Invalid Date Example

{ "error": "Invalid Date" }
